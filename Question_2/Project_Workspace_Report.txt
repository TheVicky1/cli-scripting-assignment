Observation:
A secure umask of 0077 prevents permissions from being granted to group and others by default for newly created files and directories.

5. Explanation of How Permissions Protect Project Data

- Directory mode 700 allows only the owner to enter and list directories.
- File mode 600 allows only the owner to read and modify file contents.
- Group and other users have no read, write, or execute access.
- With umask 077, future files and directories are private by default.

Together, these controls reduce accidental exposure of project plans, configuration files, data files, and logs.
